Projetos


PID para Ball and Beam


Baseado no problema "Bola Trave" este projeto foi um treinamento para o desenvolvimento de um controlador de um sistema instável em malha aberta. O sensor HC SR04 faz a leitura da distância do objeto. No centro da barra, ligado diretamente ao eixo, se encontra um micro servo 9g SG90 que altera a inclinação da barra. Com isso o carrinho passa a andar de um lado para o outro, buscando o ponto de estabilidade.


O modelo simples, traz consigo uma função de transferência simples: $$\frac{X(s)}{\Theta(s)} = \frac{2}{3}\frac{g}{s^2}$$ Se quiser saber mais, o trabalho se encontra disponível abaixo.
